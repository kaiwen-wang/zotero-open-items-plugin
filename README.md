# Open Items

Zotero plugin that opens selected items' URLs in your browser.

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+U` (Mac) / `Ctrl+Shift+U` (Win/Linux) | Open URL of selected items |

Also available via right-click > **Open URL in Browser**.

- Tries the URL field first, falls back to DOI
- Works on multiple selected items at once
- Works on attachments/notes (checks the parent item)

## Install

Download the `.xpi` from [Releases](https://github.com/kaiwen-wang/zotero-open-items-plugin/releases), then in Zotero: Tools > Add-ons > gear icon > Install Add-on From File.

Requires Zotero 7 or 8.
